About

讓 SQL 變成
真正的 API

OpenSpec 是一個 Low-Code API 平台。只需在管理介面填寫資料庫連線、SQL 語法與參數定義，系統即自動產生標準 REST API，可立即對外提供服務。

支援 PostgreSQL、Oracle、MSSQL、MySQL 等主流資料庫，內建條件區塊、計算欄位、分頁、迴圈執行等進階功能，適合快速串接各類企業資料來源。

GET /api/v1/dynamic/code/inventory-level

How It Works

三個步驟，上線

step 01
設定連線與 SQL

選擇資料庫連線，在編輯器填寫 SQL 模板。支援 SELECT / INSERT / UPDATE / DELETE / EXEC，可加入 :參數名稱 佔位符與條件區塊。

step 02
定義參數與條件

在參數頁籤定義型別、必填、預設值；在條件頁籤設定動態 WHERE 子句；在計算欄位頁籤加入後處理公式或 CASE WHEN 轉換。

step 03
發布並呼叫

點擊「發布」即上線。透過代碼路由 /code/{api_code} 或路徑路由呼叫，認證 API 以 Bearer Token 保護，可逐 API 設定。

Features

企業級功能，開箱即用

從動態 SQL 引擎到角色權限管理，OpenSpec 提供一個 API 生命週期所需的完整工具鏈——設計、測試、發布、監控，全在同一介面。

動態 SQL 引擎

支援 {{if}} 條件區塊、型別驗證與強制轉換、array 展開（IN 查詢）、JWT Claim 自動注入。

計算欄位

三種計算類型：SQL 運算式、後處理公式（回傳後執行）、CASE WHEN 條件轉換，無需修改資料庫。

多資料庫支援

PostgreSQL、Oracle（11g+ thick mode）、MSSQL、MySQL，密碼以 Fernet 加密，支援 SID / Service Name。

角色與權限

三階角色（Admin / Editor / Viewer），可逐 API 授權。HttpOnly Cookie 保護管理介面，Bearer Token 保護對外 API。

執行分析

Chart.js 儀表板：每日趨勢、P95/P99 延遲、Top 10 API、24 小時熱度，支援 7/14/30/90 天切換。

迴圈執行引擎

5 種迴圈類型：schema / date / month / list / schema_month，支援跨帳號跨月 ETL 自動 UNION ALL。

Architecture

輕量，可維護的
技術選型

  • BackendFastAPI + SQLAlchemy 2.0，async 路由、ORM 自動欄位 migration，不需要 Alembic。
  • FrontendJinja2 + HTMX + Bootstrap 5.3，無 Node.js 依賴，原生暗色模式，零打包流程。
  • EditorCodeMirror 6 via ESM CDN，SQL 語法高亮、自動完成，無需打包工具。
  • DeployDocker Compose 一鍵啟動，Oracle Instant Client 21.13 自動安裝，無需手動設定。
